### Setup API
run `npm install`  
create .env file with `DB_FILE=./db/InvoicingApp.db`  
serve on localhost:3000  
`node server`  
or install nodemon and run  
 `nodemon`  
   
 
 ### Setup frontend
 run `npm install`
 
 serve with hot reload at localhost:8080  
 `npm run dev`  
 or  
 `nodemon`
